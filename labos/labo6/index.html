Laboratoire 6: Javascript (JQuery)

Exercice 1:

En utilisant la librairie JQuery, écrire une fonction masquerElements qui prend en paramètre un sélecteur CSS et masque tous les éléments correspondants.

Exemple d'utilisation: 


<div class="monDiv">Texte 1</div>
<div class="monDiv">Texte 2</div>
<div class="monDiv">Texte 3</div>
<button onclick="masquerElements('.monDiv')">Masquer les divs</button>

Exercice 3:

En utilisant la librairie JQuery, écrire une fonction modifierStyle qui prend en paramètre un sélecteur CSS et un objet représentant les styles CSS à modifier.

Exemple d'utilisation: 


<div id="monDiv">Mon texte</div>
<button onclick="modifierStyle('#monDiv', {'background-color': 'red', 'color': 'white'})">Modifier le style</button>

Exercice 4:

En utilisant la librairie JQuery, écrire une fonction chargerContenu qui prend en paramètre une URL de fichier et charge son contenu dans un élément HTML spécifié.

Exemple d'utilisation: 


<div id="monDiv"></div>
<button onclick="chargerContenu('monFichier.txt', '#monDiv')">Charger le contenu</button>

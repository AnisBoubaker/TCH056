Laboratoire 5: Javascript

Exercice 1:

Écrire une fonction carre qui prend en paramètre un nombre et retourne son carré.

Exemple d'utilisation: 


            console.log(carre(5)); // Output : 25
            console.log(carre(3)); // Output : 9

Exercice 2:

Écrire une fonction moyenne qui prend en paramètre un tableau de nombres et retourne leur moyenne.

Exemple d'utilisation: 


            console.log(moyenne([1, 2, 3])); // Output : 2
            console.log(moyenne([4, 5, 6])); // Output : 5

Exercice 3:

Écrire une fonction changerCouleur qui prend en paramètre un élément HTML et une couleur, et qui change la couleur de fond de l'élément.

Exemple d'utilisation: 


<div id="monDiv">Mon texte</div>
<button onclick="changerCouleur(document.getElementById('monDiv'), 'red')">
        Changer la couleur en rouge</button>

Exercice 4:

Écrire une fonction supprimerElement qui prend en paramètre un élément HTML et qui supprime cet élément du document.

Exemple d'utilisation: 


<div id="monDiv">Mon texte</div>
<button onclick="supprimerElement(document.getElementById('monDiv'))">Supprimer l'élément</button>

Exercice 5:

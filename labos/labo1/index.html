Laboratoire 1: HTML

Exercice 1:

Créez une page web simple contenant un en-tête, un paragraphe et une liste à puces. Ajoutez également une image et un lien vers une autre page web.

Exercice 2:

Créez un formulaire qui permettra aux utilisateurs de saisir leur nom, leur adresse e-mail et leur âge. Ajoutez un bouton pour soumettre le formulaire.

Exercice 3:

Créez une page web avec une table qui affiche des informations sur différents produits. La table doit contenir au moins trois colonnes : le nom du produit, la description et le prix.

Exercice 4:

Créez une page web qui utilise des balises pour structurer le contenu. Utilisez des balises de titre, des balises de paragraphe et des balises de liste pour organiser le contenu de la page.

Exercice 5:

Créez une page web avec un formulaire de recherche qui permettra aux utilisateurs de rechercher des informations sur votre site web. Ajoutez également un lien vers une page d'accueil et une page de contact.
